# NI MBAYA

NI MBAYA is a Kenyan-inspired accountability app prototype that helps users stay disciplined, build streaks, and save money. The project includes a landing page with a savings calculator, daily check-in concept, and a bold local tone.

## Features

- Landing page with Kenyan branding and motivational copy
- Daily savings calculator
- Streak and progress visuals
- Modal interaction for a panic button experience
- Responsive UI built with HTML, CSS, and JavaScript

## Files

- `index.html` — main page structure and content
- `styles.css` — layout, styling, and visual design
- `script.js` — savings calculator and modal interaction
- `.gitignore` — excludes `node_modules/`

## Usage

1. Open `index.html` in your browser.
2. Enter a daily savings amount in the calculator.
3. Use the navigation links to explore the page sections.
4. Click the panic button to open the modal interaction.

## Notes

- This repo is intended as a static front-end prototype.
- `node_modules/` is excluded from source control via `.gitignore`.
